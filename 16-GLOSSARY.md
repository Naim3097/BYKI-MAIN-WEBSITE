# Module 16 — Glossary

A comprehensive list of terms used throughout the BYKI Library.

---

## A

| Term | Definition |
|------|-----------|
| **ACEA** | Association des Constructeurs Européens d'Automobiles — European oil performance classification (e.g., ACEA A3/B4, ACEA C3) |
| **AFR** | Air-Fuel Ratio — the mass proportion of air to fuel in a combustion event (stoichiometric = 14.7:1 for petrol) |
| **API** | American Petroleum Institute — grades engine oil quality (e.g., API SN, API SP, API SP Plus) |
| **ATF** | Automatic Transmission Fluid — specialised lubricant for automatic gearboxes |

## B

| Term | Definition |
|------|-----------|
| **BLE** | Bluetooth Low Energy — the wireless protocol used by modern OBD-II adapters to communicate with phones/computers |
| **BYKI** | The vehicle health diagnostics platform you're reading about |
| **Bank 1 / Bank 2** | Refers to the two cylinder banks in V-configuration engines. Bank 1 contains cylinder #1. |

## C

| Term | Definition |
|------|-----------|
| **CAN Bus** | Controller Area Network — the primary vehicle communication protocol used in all OBD-II vehicles from 2008+ |
| **Catalyst / CAT** | Catalytic converter — exhaust component that converts toxic gases (CO, HC, NOx) into less harmful emissions |
| **Coolant Temperature** | Engine Coolant Temperature (ECT) — normally 85–105°C when fully warmed up |
| **CVT** | Continuously Variable Transmission — a type of automatic transmission that uses a belt/chain instead of fixed gears |

## D

| Term | Definition |
|------|-----------|
| **DCT** | Dual-Clutch Transmission — an automated manual transmission with two clutch packs for faster gear changes |
| **DLC** | Diagnostic Link Connector — the standardised 16-pin OBD-II port in your vehicle |
| **DTC** | Diagnostic Trouble Code — a standardised fault code stored by the ECU when a malfunction is detected (e.g., P0300) |
| **Drive Cycle** | A specific sequence of driving conditions (start, idle, accelerate, cruise, decelerate) required for the ECU to run its self-tests |

## E

| Term | Definition |
|------|-----------|
| **ECU** | Engine Control Unit (also Electronic Control Unit) — the computer that manages engine operation |
| **EGR** | Exhaust Gas Recirculation — system that recirculates a portion of exhaust gas back into the intake to reduce NOx emissions |
| **ELM327** | A popular OBD-II interpreter chip used in many aftermarket Bluetooth adapters |
| **EVAP** | Evaporative Emission Control System — prevents fuel vapours from escaping into the atmosphere |

## F

| Term | Definition |
|------|-----------|
| **Freeze Frame** | A snapshot of sensor data captured by the ECU at the moment a DTC was triggered |
| **Fuel Trim** | The ECU's adjustment to fuel injection to maintain the ideal air-fuel ratio. Short-Term (STFT) reacts instantly; Long-Term (LTFT) is a learned correction. |

## H

| Term | Definition |
|------|-----------|
| **Health Score** | BYKI's 0–100 overall vehicle health rating derived from sensor analysis and fault code assessment |
| **Health Scan** | The comprehensive multi-phase vehicle analysis performed by BYKI |

## I

| Term | Definition |
|------|-----------|
| **ILSAC** | International Lubricants Standardization and Approval Committee — sets oil performance standards (e.g., GF-5, GF-6A, GF-6B) |
| **Intake Air Temperature** | IAT — temperature of air entering the engine; typically 35–55°C in Malaysian climate |
| **ISO 9141** | An older OBD-II communication protocol used in some European and Asian vehicles pre-2008 |

## J

| Term | Definition |
|------|-----------|
| **J1850** | SAE J1850 — an older OBD-II protocol (PWM or VPW variants) used primarily in older American vehicles |
| **J2012** | SAE J2012 — the standard defining DTC structure and numbering |

## K

| Term | Definition |
|------|-----------|
| **KWP2000** | Keyword Protocol 2000 (ISO 14230) — an OBD-II communication protocol used in some Asian and European vehicles |
| **Knock Sensor** | Detects engine detonation (knock) and signals the ECU to retard ignition timing |

## L

| Term | Definition |
|------|-----------|
| **Lambda** | Ratio of actual AFR to stoichiometric AFR. Lambda 1.0 = perfect for petrol. Lambda > 1 = lean. Lambda < 1 = rich. |
| **Live Data** | Real-time sensor values streamed from the vehicle's ECU via OBD-II |
| **LTFT** | Long-Term Fuel Trim — the ECU's learned fuel correction factor. Normal range: ±10%. |

## M

| Term | Definition |
|------|-----------|
| **MAF** | Mass Air Flow sensor — measures the mass of air entering the engine (grams/second) |
| **MAP** | Manifold Absolute Pressure sensor — measures intake manifold pressure (kPa) |
| **MIL** | Malfunction Indicator Lamp — the "check engine" light on your dashboard |
| **Mode** | OBD-II service mode — Mode 01 (live data), Mode 02 (freeze frame), Mode 03 (read DTCs), Mode 04 (clear DTCs), etc. |

## O

| Term | Definition |
|------|-----------|
| **O2 Sensor** | Oxygen sensor — measures oxygen content in exhaust gas. Used by the ECU to adjust fuel mixture. |
| **OBD-II** | On-Board Diagnostics, 2nd generation — the standardised vehicle diagnostic system mandated in all modern vehicles |
| **OEM** | Original Equipment Manufacturer — the vehicle maker (e.g., Toyota, Proton, BMW) |

## P

| Term | Definition |
|------|-----------|
| **Pending DTC** | A fault code that has been detected once but not yet confirmed across multiple drive cycles |
| **Permanent DTC** | A fault code that cannot be cleared manually — only the ECU can remove it after the underlying issue is resolved |
| **PID** | Parameter ID — a code used to request a specific data value from the ECU (e.g., PID 0x0C = Engine RPM) |

## R

| Term | Definition |
|------|-----------|
| **Readiness Monitors** | Self-test routines run by the ECU (e.g., catalyst, EVAP, O2 sensor). Status: Complete, Incomplete, or Not Supported. |
| **RPM** | Revolutions Per Minute — engine crankshaft rotational speed. Normal idle: 650–850 RPM. |

## S

| Term | Definition |
|------|-----------|
| **SAE** | Society of Automotive Engineers — the standards body responsible for OBD-II, DTC numbering, and many automotive protocols |
| **Sensor** | An electronic component that measures a physical quantity (temperature, pressure, speed, etc.) and reports it to the ECU |
| **Stoichiometric** | The chemically ideal air-fuel ratio for complete combustion — 14.7:1 for petrol engines |
| **STFT** | Short-Term Fuel Trim — the ECU's instantaneous fuel correction. Normal range: ±10%. |
| **Syntium** | Petronas' premium engine oil brand, available in multiple tiers from mineral to fully synthetic |
| **System Score** | A 0–100 rating for one of the seven systems evaluated by BYKI (Combustion, Fuel, Cooling, Emission, Charging, Oil, Transmission) |

## T

| Term | Definition |
|------|-----------|
| **TPS** | Throttle Position Sensor — reports how far the throttle is open (0–100%) |
| **Thermostat** | A valve in the cooling system that opens at a set temperature (typically 82–90°C) to allow coolant flow to the radiator |
| **Timing Advance** | The number of degrees before top dead centre (TDC) at which the spark plug fires |
| **Torque Converter** | The fluid coupling between the engine and automatic transmission that allows the engine to idle while the vehicle is stopped |
| **Turbocharger** | A forced-induction device driven by exhaust gases that compresses intake air for increased engine power |

## V

| Term | Definition |
|------|-----------|
| **VIN** | Vehicle Identification Number — a unique 17-character code identifying your specific vehicle |
| **Viscosity** | A measure of oil's resistance to flow. Format: XW-YY (e.g., 5W-30). W = winter rating, second number = high-temperature rating. |
| **Voltage (Battery)** | Normal: 12.4–12.8V engine off, 13.5–14.7V engine running. Below 12.0V indicates a weak battery. |

## W

| Term | Definition |
|------|-----------|
| **Web Bluetooth** | A browser API that allows web applications to communicate with Bluetooth devices — the technology BYKI uses to connect to your OBD-II adapter |
| **Workshop** | In BYKI context, a BYKI Partner Workshop — a service centre that accepts BYKI health reports and stocks Petronas Syntium products |
